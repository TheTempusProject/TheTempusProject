The Tempus Project

PHP powered web application for rapid prototyping

Powerful

The Tempus Project is built with expansion in mind. From a custom template engine to the simple to use MVC style, The Tempus Project is built to provide a powerful and stable foundation for web applications.

Quick and Simple Installation

Built with rapid deployment in mind you can have The Tempus Project installed in just minutes.

Open Source

The Tempus Project is completely open source and only utilizes other open-source components. The Project is provided under thee MIT license.

This is the home of The Tempus Project, a web application and framework built on php. The Tempus Project is designed to be easily installed, managed, and expanded, but the most important feature is the expandability.

You can easily utilize existing functionality to build your own custom website. The application is written in a way that allows you to very easily add functionality. So if you can think of it, you can easily build it into the existing framework. Whether you're an Industry pro, a freelancer, or even a novice the aim of the project is to help you build powerful customized websites.

What we are trying to provide is a veritable crash course in web development centered around this framework. Regardless of you skill level or prior knowledge, the goal is to help you start building web applications.

Currently the crash course is still being built and tested but we already have guides available for How to Setup a Web Server and How to Install The Tempus Project. The public documentation is expanding daily and we are accepting users for the first round when the crash course is completed. You can find more information at Crash Course Signup.
